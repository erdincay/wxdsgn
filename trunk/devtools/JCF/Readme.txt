

1.Download JediCodeFormat from http://JediCodeFormat.sourceforge.net, extract it. 
2.Place "JCFSettings.cfg" into the JediCodeFormat directory.
3.Make sure there's no mistake in your source files. JCF will stop if there's an error.
4.Run JCF.

Note:  When you select "All files in directory and all subdirectories", 
      please input the "Start directory" Manually. Browse will not work.
