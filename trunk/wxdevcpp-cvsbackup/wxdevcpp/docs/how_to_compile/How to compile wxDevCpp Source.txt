Apart from the standard dev-cpp components
You need the following component packs to compile wxdevcpp source.
You can also find a copy of the components in the directory "CompUsedBy_WX_BUILD"

1) Latest snapshot of JVCL 3.0 http://jvcl.sf.net  
2) Latest snapshot of JCL  beta http://jcl.sf.net

(use CompUsedBy_WX_BUILD\jvcl\packages\JVCL3Install.exe to do 
a automatic installation ofjcl/jvcl)

3) EXtLib from http://www.Torry.net
4) ThemeManager from http://www.delphi-gems.com/
