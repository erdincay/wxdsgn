Hello,
wx-Devcpp is an extension of DevC++ by Colin Laplace Et.al. This program helps you to create Dialogs for wxWidgets visually. With all the good features of dev-cpp, wx-DevCpp hosts a wxWidgets form designer with the latest wx library. The main intension of this program is to fill the gap of the non-availability of a commercial grade IDE for the wxWidgets development.

Beta 6.6 is released today. Although there is no big change in this release, there are quite a lot of bug fixes went in to it. If you are using the previous version of wx-devcpp, please uninstall and install the new version.
Thanks to Tony ( reina_ga AT hotmail DOT com) making a rock solid setup.

Support wxWidgets

If you use wx-DevCpp in commercial project and find it to be useful in your work, I request you to donate couple of bucks to the wxWidgets development community. Your contribution to wxWidgets project will directly contribute to the quality and future bug fixes for wx-Devcpp

wxWidgets need lot of funding to develop further, so feel free to send your donation to Julian Smart. Please note even a small micropayment via paypal will be a great help.

Check wxWidgets donation page at http://wxwidgets.org/spi.htm
