Adding and Removing files


Adding and removing files is very easy.

You can add multiple files in the same time by clicking on Project menu, then on Add to Project.


You have different ways for removing files from your project.

Either click on Project menu, then on Remove from Project and select the file you want to remove in the list, or right-click on the file you want to remove in the Project Manager window, and click on Remove file
