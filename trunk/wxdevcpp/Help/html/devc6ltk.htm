Step 2

Type a name for your project in the next dialog, like "Project 1" and click on OK.

You will have then to save your project in a directory. You can save it for example in c:\My Documents\ and then click on Save. All the necessary files will be created in the directory you selected and your project will be opened in Dev-C++, with a new source file inside (the code inside it is the base for a win32 program).
