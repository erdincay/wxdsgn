Code tab


   
    This is supposed to allow the user to define a menu option that adds code snippets to the IDE menu.  The idea is that if you have code that you frequently use (e.g. a class or function that you almost always add to your programs), then you can set up a menu selection that automatically adds this code to your current project.  This doesn't seem to be working anymore. I'm not sure if it's useful to anyone anyway.
